Files encrypting / decrypting

This tool has 2 ways of saving file after encrypting and decrypting :
- v1 : for all browsers.
The final file is stored in memory before offering it for download, which implies that the maximum size of a file to be encrypted/decrypted cannot exceed the RAM memory available on your device.
Maximum authorized size: 5Gb.
If the % progress no longer changes, it is probably because you do not have enough memory available at the moment.
Before trying again, to free up memory, close the most demanding applications as well as tabs in your browser.

- v2: for some browsers (see the list):
This version uses an API that allows you to save the file on your device as you go and therefore does not encounter the limiting memory problem of v1.
So prefer v2 if possible by using one of the supported browsers (as of 01/26/2025: Chrome >= 86, Edge >= 86, Opera >= 72, Chrome Android >= 132).
